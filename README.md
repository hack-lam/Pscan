# Pscan 端口扫描工具
多线程端口扫描工具 | 支持 Top100 / Top1000 / Web端口 / 全端口扫描

## 工具介绍
Pscan 是一款基于 Python3 开发的轻量多线程端口扫描工具，无需第三方依赖，内置 4 种扫描模式，可快速检测目标 IP 的开放端口，适用于内网资产探测、安全测试、端口巡检。

## 版本信息
- 工具名称：Pscan
- 当前版本：v2.0.0
- 开发语言：Python3
- 依赖库：socket、threading（均为系统内置，无需安装）

## 功能特点
- 多线程高并发扫描，速度可调
- 内置 4 种端口扫描模式
- 彩色终端输出，开放/关闭端口一目了然
- 扫描完成自动统计开放端口列表
- 无第三方依赖，开箱即用
- 支持 Windows / Linux / macOS 全平台

## 端口模式说明
1. Top100 端口：高频常用端口（80、443、22、3389 等）
2. Top1000 端口：渗透测试常用全量端口
3. 常见 Web 端口：网站、中间件、管理后台端口
4. 全端口：1~65535 完整端口扫描
